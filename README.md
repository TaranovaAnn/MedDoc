# МИС «МедДок»

Веб-приложение для автоматизации работы поликлиники: ведение карт пациентов, запись на приём, расписание врачей.

## Стек
- Python 3 + Flask
- Jinja2 + HTML/CSS

## Запуск
```bash
pip install flask
python app.py
```
Открыть: http://127.0.0.1:5000
